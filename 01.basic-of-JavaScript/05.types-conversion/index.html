Type conversion in JavaScript

Type conversion, also known as type casting, refers to the process of converting a value from one data type to another in JavaScript. JavaScript provides several methods and operators for performing type conversion. Here are some common ways to convert data types:

    List of JavaScript Type conversion.

  • 1. String Conversion
  • 2. Number Conversion
  • 3. Boolean Conversion
  • 4. Implicit Type Conversion

  • String Conversion: To convert a value to a string, you can use the toString() method or the String() function.

    For example:

                  
  ar number = 42;
  var string = number.toString();   // "42"
  var anotherString = String(number);   // "42"

  • Number Conversion: To convert a value to a number, you can use the parseInt() or parseFloat() functions, or the unary plus (+) operator.

    For example:

                  
  var string = "42";
  var number = parseInt(string);    // 42
  var anotherNumber = +string;    // 42

  • Boolean Conversion: To convert a value to a boolean, you can use the Boolean() function or apply a logical operator like !! (double negation).

    For example:

                  
  var value = "Hello";
  var boolean = Boolean(value);    // true
  var anotherBoolean = !!value;    // true

  • Implicit Type Conversion: JavaScript also performs implicit type conversion automatically in certain situations, such as when using arithmetic operators with different data types.

    For example:

                  
  var result = 5 + "3";    // "53" (implicit conversion of number to string)
  var total = "10" - 5;    // 5 (implicit conversion of string to number)

It's important to be aware of the potential issues and unexpected behavior that can arise from type conversions, especially with implicit conversions. It's often a good practice to explicitly convert data types when needed to ensure the desired results.
