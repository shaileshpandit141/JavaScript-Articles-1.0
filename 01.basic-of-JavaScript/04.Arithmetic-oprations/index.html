Arithmetic operators in JavaScript

Arithmetic operators are used in JavaScript to perform mathematical operations on numeric values. Here are the common arithmetic operators in JavaScript:

    List of JavaScript Arithmetic operator

  • 1. Adition
  • 2. Subtraction
  • 3. Multiplication
  • 4. Division
  • 5. Remainder/Modulus
  • 6. Exponentiation/power

  • Addition (+): Adds two values together or concatenates two strings.

    For example:

                
  var sum = 5 + 3;     // 8
  var fullName = "John" + " Doe";    // "John Doe"

  • Subtraction (-): Subtracts one value from another.

    For example:

                
  var difference = 10 - 5;    // 5

  • Multiplication (*): Multiplies two values.

    For example:

                
  var product = 2 * 3;    // 6

  • Division (/): Divides one value by another.

    For example:

                  
  var quotient = 10 / 2;    // 5

  • Remainder/Modulus (%): Returns the remainder after division.

    For example:

                  
  var remainder = 10 % 3;    // 1

  • Exponentiation/power (**):Raises a number to the power of another number.

    For example:

                  
  var result = 2 ** 3;    // 8 (2 raised to the power of 3)

    • These arithmetic operators can be used with numeric values to perform calculations and manipulate data in JavaScript.
