JavaScript Data Types

In JavaScript, there are several data types that variables can hold. Here are the commonly used data types in JavaScript:

    JavaScript Datatypes List

  • 1. Number
  • 2. String
  • 3. Boolean
  • 4. Null
  • 5. Undefined
  • 6. Object
  • 7. Array
  • 8. function
  • 9. Symbol
  • 10. And more...

  • Number: Represents numeric values, including integers and floating-point numbers.

    For example:

                
  let age = 25;

  • String: Represents textual data enclosed in single or double quotes.

    For example:

                
  let name = 'mango';

  • Boolean: Represents a logical value, either true or false.

    For example:

                
  let isActive = true;

  • Null: Represents the intentional absence of any object value. It is a special value that indicates the absence of a value.

    For example:

                
  let data = null;

  • Undefined: Represents an uninitialized or unassigned variable. When a variable is declared but not assigned a value, it is automatically assigned the value undefined.

    For example:

                
  let x;

  • Object Represents a collection of key-value pairs, where each value can be of any data type. Objects are denoted by curly braces {}.

    For example:

                
  let person = {
    name: "mango",
    age: 25,
  };

  • Array: Represents an ordered collection of values. Arrays are denoted by square brackets [].

    For example:

                
  let numbers = [1, 2, 3, 4, 5, 'mango'];

  • Funaciton: Represents reusable blocks of code that can be invoked or called to perform a specific task. Functions can be assigned to variables or defined separately.

    For example:

                
  function greet(name) {
    console.log("Hello, " + name + "!");
  }

  • Symbol: Represents a unique identifier. Symbols are often used as keys in objects to avoid naming conflicts.

    For example:

                
  let id = Symbol("uniqueId");

These are the main data types in JavaScript. Understanding data types is important for properly handling and manipulating data in JavaScript programs.
